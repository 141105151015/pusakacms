# Kenalan PusakaCMS adalah Content Management System berbasis file, dalam artian semua konten disimpan di dalam file teks tanpa menggunakan database server seperti MySQL, PostgreSQL atau SQLite. PusakaCMS mengutamakan kesederhanaan dalam proses penyimpanan dan publikasi konten, serta kecepatan akses dan kerampingan sistem. Kamu tidak akan menemukan proses instalasi seperti CMS lain atau konfigurasi database di dalam sistem PusakaCMS.

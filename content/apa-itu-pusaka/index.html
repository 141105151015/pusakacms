# Perkenalan **PusakaCMS** adalah *Content Management System (CMS)* berbasis file, dalam artian semua konten disimpan di dalam file teks tanpa menggunakan *database engine* seperti MySQL, PostgreSQL atau SQLite. PusakaCMS mengutamakan kesederhanaan dalam proses penyimpanan dan publikasi konten, serta kecepatan akses dan kerampingan sistem. Kamu tidak akan menemukan proses tahapan instalasi atau konfigurasi database di dalam sistem PusakaCMS. Bukan berarti CMS dengan *Database Management System (DBMS)* berjalan lebih lambat dan berat dan bukan berarti CMS berbasis file selalu lebih cepat. Kecepatan proses juga dipengaruhi oleh berbagai faktor seperti spesifikasi server, selain tentunya optimasi kode program. PusakaCMS dibangun di atas framework **[CodeIgniter](http://ellislab.com/codeigniter)** yang terkenal unggul dalam hal kecepatan proses dan kemudahan pengembangan. Dengan dua faktor ini (konten berbasis file dan framework CodeIgniter), PusakaCMS diharapkan dapat menjadi solusi bagi Kamu jika Kamu punya satu atau lebih dari kriteria berikut: - Kamu penulis yang hanya ingin **fokus pada konten**, dan ga mau ribet sama kodingan dan fitur macam-macam - Kamu memerlukan pembangunan **dokumentasi yang simpel, cepat dan ringan** untuk aplikasi atau proyek di bidang yang sedang kamu kerjakan - Kamu **programmer CodeIgniter** dan memerlukan managemen konten yang sederhana dan mudah dikembangkan untuk proyek selanjutnya - Kamu perlu membuat **web statis dengan budget ekonomis**, untuk keperluan pribadi ataupun klien - Kamu sudah __akrab dengan *Versioning Control System (VCS)*__ semisal **Git**, **SVN** dan **Mercurial** dan ingin merekam revisi kontenmu dengan itu - Kamu ingin menulis buku, diary atau *blogging* dengan cara yang baru dan tidak biasa PusakaCMS hampir tidak memerlukan proses konfigurasi dan proses instalasi untuk dapat *online*. Kamu tinggal unggah dan ekstrak di hostingan kamu, dan Kamu sudah bisa mulai mengisi konten di web barumu. Sesederhana itu.
